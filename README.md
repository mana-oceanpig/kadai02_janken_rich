# ①課題番号-プロダクト名

猫セラピスト

## ②課題内容（どんな作品か）

- 猫のオススメのセラピー習慣を教えてもらえるアプリ

## ④工夫した点・こだわった点

- ホームページと次のページでhtmlを分けた点
- ユーザビリティに配慮してリセットボタンを実装した点
- デイリークエスト化させた点

## ⑤難しかった点・次回トライしたいこと(又は機能)

- 本当は習慣ごとに難易度を付与したい
- 難易度毎にメッセージを変化させたい
- 難易度毎に好感度ポイントを付与し、「整えたゲージ」を実装したい
